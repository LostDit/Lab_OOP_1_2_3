### Задача 2. Класс Fifteen
- Закрытые поля: `Field game_field; и int count;`

- Открытые: конструктор `Fifteen(size_t size);, деструктор ~Fifteen();`

- Метод `int getCount() const;`

- Метод `void run();` — перемешивает поле, обнуляет счётчик

- Метод `void draw() const;` — вывод поля в виде таблицы (числа через пробел)

- Метод `bool isGameOver() const;` — проверка соответствия начальной расстановке

- Метод `void onKeyPressed(int const& btnCode);` — обработка нажатий (коды 72,80,75,77), увеличивает счётчик при перемещении

- Запретить копирование и присваивание (как в `Field`)

- Реализовать игровой цикл в `main` (создание, `run()`, отрисовка, обработка клавиш до победы)

### Задача 3. Класс Statistic (перегрузка конструкторов и оператора присваивания)
- Конструктор от массива: `Statistic(int const * a, size_t const & n);`

- Конструктор от строки: `Statistic(std::string const & str);`

- Конструктор копирования: `Statistic(Statistic const& statistic);`

- Оператор присваивания: `Statistic& operator =(Statistic const& statistic);`
