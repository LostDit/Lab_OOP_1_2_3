### *Чеклист:* 
- [x] 1. Создать файлы "Employee.h" и "Employee.cpp" для описания в них записи и перегрузки операторов вывода << и ввода >>.  

- [x] 2. Заполните массив (не менее 8 элементов) из структур с клавиатуры. Запишите массив в бинарный файл, в текстовый файл. 

- [x] 3. (хорошо) Начальные данные для формирования массива брать из бинарного файла.

- [x] 4. Все задания по выборке элементов выполнить для массива структур. Результаты выборки выводить в текстовые файлы, а также дублировать вывод на экран.

- [x] 5. Программа должна иметь пользовательский интерфейс с возможностью выбора действий. 

- [x] 6. (отлично) Изменить массив на вектор и добавить соответствующие перегрузки функций.
